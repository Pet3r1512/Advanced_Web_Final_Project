--------------Hướng dẫn setup project------------------------
*Bằng cách tải project

    Bước 1: Tải project từ Github hoặc giải nén tư mục source
    Bước 2: Mở Terminal và chạy lệnh "npm install" để tải các module và package được sử dụng trong dự án
    Bước 3: Tại các file admin.route.js, emailValidation.js và phoneValidation.js (nếu sử dụng Visual Studio Code có thể gõ tổ hợp phím "Ctrl P" và nhập các tên file như trên), hãy thay đổi các link để gọi api như hướng dẫn tại các file đó và lưu lại
    Bước 4: Mở Terminal và chạy lệnh "npm run dev" để khởi chạy dự án. Lúc này dự án sẽ truy cập đường thông qua đường dẫn "localhost:3000"

*Bằng tên miền
    Có thể truy cập đường link sau để đi đến trang web của dự án được deploy trên RailwayApp. 

    https://paywind.up.railway.app



------------Hướng dẫn đăng nhập với vai trò Admin-----
    Bước 1: Hoàn thành setup dự án hoặc truy cập vào tên miền ở trên
    Bước 2: Truy cập vào trang Đăng nhập (Sign In) 
    Bước 3: Sử dụng tài khoản là "admin" và mật khẩu là "123456"
